# نظام إدارة المشاريع العربي (النسخة الاحترافية v5)

تطبيق ويب متطور لإدارة المشاريع والمهام، يستغل كافة إمكانيات قاعدة البيانات ويوفر تجربة إدارية ذكية.

## 🚀 الميزات الجديدة (التحديث v5)

### 1. استغلال كامل لقاعدة البيانات
- **المهام:** إضافة الوصف، تاريخ البداية، نسبة التقدم التفاعلية (0-100%)، الوسوم (Tags)، والمهام الإضافية (Extra Tasks).
- **المشاريع:** إضافة الأولوية (Priority) والحالات المتقدمة.
- **سجل الأنشطة (Activity Log):** تتبع حركات المستخدمين (إنشاء، تحديث، حذف) وعرضها في لوحة التحكم.

### 2. إصلاحات الرؤية والوصول (الحل النهائي)
- تم إصلاح استعلامات الإسناد؛ الآن تظهر المهام فوراً للمستخدم المسند إليه بدقة متناهية.
- تم فلترة قائمة المستخدمين لاستبعاد "مدير النظام" من خيارات إسناد المهام، ليبقى التركيز على فريق العمل.

### 3. تجربة مستخدم موحدة (UI/UX)
- توحيد القائمة الجانبية (Sidebar) في جميع الصفحات (التقويم، التحليلات، المهام) لضمان سهولة التنقل.
- إضافة شريط تقدم (Progress Bar) لكل مهمة ولكل مشروع.
- تحسين نظام التعليقات ليظهر دور المعلق (مدير/عضو).

### 4. تحليلات متقدمة
- لوحة تحليلات الفريق تعرض تقييماً ذكياً لكل عضو (ممتاز، جيد، مستقر، يحتاج متابعة) بناءً على الأداء الفعلي.

## 🛠️ الهيكل التقني
- **supabase-client.js:** النسخة v5 المطورة لدعم كافة الحقول الجديدة وسجل الأنشطة.
- **pages/my-tasks.html:** واجهة محدثة لإدارة المهام الشخصية بنسبة التقدم.
- **pages/team-analytics.html:** واجهة تحليلية شاملة للمدراء.

## 🔐 كيفية البدء
1. افتح `pages/index.html`.
2. سجل الدخول بحسابك.
3. استمتع بتجربة إدارية متكاملة.
